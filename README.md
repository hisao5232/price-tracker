# Price Tracker & Search Tool

インターネットサイトの商品の価格推移を追跡し、新着商品を素早く検索するためのウェブアプリケーションです。
自身の副業およびプログラミング学習の一環として、モダンな技術スタック（Next.js + FastAPI）を用いて構築しています。

## 🚀 主な機能

- **商品検索**: キーワードによる新着商品のリアルタイム検索。
- **価格追跡**: 指定した商品の価格を定期的にチェックし、履歴を保存（実装中）。
- **価格通知**: 設定した目標価格を下回った際の通知機能（予定）。

## 🛡️ スクレイピングに関するポリシー

本ツールは、対象サイトへの負荷を最小限に抑え、持続可能なデータ取得を行うことを最優先に設計しています。

- **負荷軽減**: リクエスト間に適切な待機時間を設け、短時間での大量アクセスを回避しています。
- **効率的な取得**: 必要なデータのみをピンポイントで取得し、サーバーリソースを無駄に消費しません。
- **規約への配慮**: ヘッドレスブラウザの挙動を最適化し、サイト側に過度な負担をかけない「紳士的なスクレイピング」を遵守しています。

## 🛠 技術スタック

- **Frontend**: Next.js (App Router), Tailwind CSS v4, React Compiler
- **Backend**: FastAPI (Python), Playwright
- **Database**: PostgreSQL (v16)
- **Infra**: Docker, Traefik (Reverse Proxy)
- **OS**: Ubuntu 24.04 (VPS)

## 📦 セットアップ

1. リポジトリをクローン
2. `.env` ファイルを作成（サンプルを元に設定）
3. Dockerで起動:
   ```bash
   docker compose up --build -d
   ```
